# 🔍 Nmap - My Notes

## What is Nmap?
Nmap is a network scanner tool. Used to find open ports, services and vulnerabilities.

## Why Hacker Use Nmap?
To find weakness in target system before attack.

## Basic Commands I learned  
## Common Ports
- 22 = SSH
- 80 = HTTP (Website)
- 443 = HTTPS (Secure Website)
- 21 = FTP
- 25 = SMTP (Email)
- 3306 = MySQL Database
